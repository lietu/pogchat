PogChat

This is a simple tool to allow you to view a Twitch chat without Flash.

The purpose of this tool is to give streamers the ability to put the Twitch chat on any and all tablet, phone, Raspberry Pi, etc. and other devices that may not support Flash.

You will NOT be able to actively participate in the chat, and there is no way to moderate.

That said, if you're interested, fill in the channel name (and fiddle with the extra settings if you feel like it) below and go wild.

Copying the URL after that will bring you to the same chat without having to re-enter things here.

Settings are also automatically saved as the global defaults when starting the chat using local storage (similar to cookies).
